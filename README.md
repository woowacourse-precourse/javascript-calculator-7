## 🔧 구현할 기능

1. 사용자 입출력
2. 입력문자열 파싱 및 계산 로직 구현
3. 커스텀 구분자 파싱 및 계산 로직 구현
4. 입력 예외 처리
5. 테스트 케이스 적용
