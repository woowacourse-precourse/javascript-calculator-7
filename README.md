# javascript-calculator-precourse

## 구현할 기능 목록

### 1. 사용자 입력 처리

- [x] 사용자의 문자열 입력받기

### 2. 입력 문자열 파싱

- [x] 입력이 없다면 빈 문자열 리턴
- [x] 정규 표현식에 기본 구분자 추가
  - [x] `,`(쉼표) 또는 `:`(콜론)
- [x] 정규 표현식에 커스텀 구분자 추가
  - [x] `//`로 시작해서 `\n`으로 끝나는지 확인
  - [x] 올바른 형식이 아닐 경우 에러 처리
- [x] 정규 표현식을 바탕으로 구분자와 숫자 문자열 분리
- [x] 숫자 없이 구분자만 입력된 경우 에러 처리
- [x] 유효하지 않은(커스텀 구분자 이외의) 문자 에러 처리

### 3. 숫자 유효성 검사

- [x] 입력된 값이 양수인지 확인

### 4. 값 계산

- [x] 파싱된 숫자들의 합 계산
- [x] 빈 문자열 입력 시 0 반환

### 5. 계산 결과 출력

- [x] 값 계산의 결과 출력

### 추가 고려사항

- [ ] 모든 함수의 단위 테스트 작성
- [x] 코드 스타일 및 컨벤션 준수
