# javascript-calculator-precourse

**프로젝트 설명: 입력한 문자열에서 숫자를 추출하여 더하는 계산기**

## 기능 요구 사항
- 문자열을 입력받아 특정 구분자로 숫자를 분리
- 분리된 숫자들을 더해서 결과 반환
- 예외 처리

## 주요 기능 분리
- 입력 받기: 문자열 입력
- 구분자 설정: 기본 구분자(',', ':'), Custom 구분자 설정
- 문자열 분리: 설정된 특정 구분자(Default 구분자 + Custom 구분자)로 숫자 분리
- 숫자 합산: 분리된 숫자들의 합 반환
- 에러 처리: 양수가 아닌 값이 들어올 경우 예외 처리

## 입출력 요구 사항
> **입력**: 구분자와 양수로 구성된 문자열  
> **출력**: 덧셈 결과  
>   
> **실행 결과 예시**  
>   
> 덧셈할 문자열을 입력해 주세요.  
> `1,2:3`  
> 결과 : `6`
