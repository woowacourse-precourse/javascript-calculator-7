# javascript-calculator-precourse
## 🛠 구현할 기능 목록

1. @woowacourse/mission-utils 에서 제공하는 Console 로 입출력 

2. 커스텀 구분 문자열 제작
- 커스텀 문자열이 여러개 올 수 있음

3. 문자열을 바탕으로 숫자의 합 계산

4. 예외 케이스 생각하여 [ERROR] 발생 후 애플리케이션 종료
- 양수가 아닐 경우
- 구분자가 아닐 경우
- 커스텀 문자열 자리에 숫자가 올 경우
- 커스텀 문자 지정할 때 // \n으로 하지않는경우
- 숫자 - 문자 - 숫자 형태를 유지하지 않을 때
