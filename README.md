# javascript-calculator-precourse

## 🛠️ 구현할 기능과 목록
1. 문제 출제 및 사용자 값 입출력 구현
2. 계산기 구현
- 쉼표 또는 콜론이 구분자로 구분되어 수가 들어오는 구분자를 기준으로 분리한 수를 더한다.
- 커스텀 구분자 : `//`와 `\n` 사이에 위치하는 문자 <br>
예시: `//;\n1;2;3` 일경우 구분자는 `;`
- 에러 처리: 사용자가 잘못된 값을 입력할 경우 "[ERROR]" 메시지와 함께 Error를 발생시킨 후 애플리케이션 종료
3. 입출력 형식 맞추기
