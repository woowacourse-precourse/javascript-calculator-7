# javascript-calculator-precourse

## 기능 목록

- 사용자 입력 받기
  - "덧셈할 문자열을 입력해 주세요."
- 커스텀 구분자 존재 여부에 따른 구분자 설정
  - 커스텀 구분자 / 쉼표(,) 또는 콜론(:)
- 구분자를 기준으로 순자 분리
- 분리된 숫자 덧셈
- 결과 값 출력
- 잘못된 값 입력 시 에러 발생
  - 지정된 구분자(커스텀 구분자) 외 다른 문자가 존재하는 경우
  - 지정된 구분자(기본 구분자) 외 다른 문자가 존재하는 경우
  - (//)와 (\n) 사이 구분자가 여러 글자인 경우
  - (//)와 (\n) 사이 구분자가 없는 경우
  - (//)가 빠진 경우
  - (\n)가 빠진 경우
  - (//)의 위치가 잘못된 경우
