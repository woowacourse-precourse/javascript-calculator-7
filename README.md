# javascript-calculator-precourse

## 🔢 문자열 덧셈 계산기 -  구현할 기능 목록

- 문자열을 요구하는 문장 출력하기
- 문자열 입력받기
- 커스텀 구분자가 있는지 확인하기
    - 문자열 앞부분의 "//"와 "\n" 확인
- 기본 구분자(쉽표(,), 콜론(:))와 커스텀 구분자를 기준으로 숫자 추출하기
- 잘못된 입력인지 확인
    - 구분자와 양수이외의 입력이 있다면 에러 발생시킨 후 애플리케이션 종료하기
    - "[ERROR]" 로 시작하는 메세지와 함께 에러 발생
- 추출한 숫자들 더하기
- 결과 출력하기
