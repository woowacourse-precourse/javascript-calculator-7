# javascript-calculator-precourse

> 기능
- 유저에게 덧셈할 문자열을 입력받는다.
- 쉼표(,)나 콜론(:)을 기본 구분자로 인식한다.
- 문자열 앞부분의 "//"와 "\n" 사이 문자를 커스텀 구분자로 인식한다.
- 구분자를 기준으로 숫자를 분리한다.
    - 예1) "" => 0, "1,2" => 3, "1,2:3" => 6  
    - 예2) "//;\n1;2;3" => 6
- 분리한 숫자들을 전부 더한다.
<br>

> 예외 처리
- 문자열에 구분자와 양수 외의 문자가 포함됐는지 판단한다.
    - 그럴 경우 "[ERROR]"로 시작하는 메시지를 띄운다.
    - Error를 발생시킨 후 애플리케이션을 종료한다.
