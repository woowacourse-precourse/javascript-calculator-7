## 📝 구현할 기능 목록

### 사용자 입력

- [x] 문자열을 입력받는다.

### 문자열 앞부분 커스텀 구분자 추출

- [x]  커스텀 구분자를 쓴다면 '// \n'가 제일 앞에 오는지 확인한다.
- [x] '// \'n사이의 커스텀 구분자 추출한다.
- [x] 추출 이후 '// \'n은 문자열에서 제거한다.

### 입력 유효성 검사

- [x] 문자열 길이가 0이면 통과시킨다.
- [x] 문자열에서 [,] [;] [커스텀구분자] [숫자문자]를 제외한 문자가 있는지 확인한다.

### 구분자에 따라 문자열을 파싱해서 숫자 배열 추출하기

- [x] 구분자를 사이의 숫자 문자를 파싱해낸다.

### 추출된 양수 sum변수에 더해주기

- [x] 추출해낸 숫자를 다 더해준다.

### 출력

- [x] 오류 없이 덧셈이 되었다면 덧셈 결과를 출력한다.
